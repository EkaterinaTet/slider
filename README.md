## Описание 
1. Реализация самой линии слайдера.
2. Возможность drag мышкой и свайпать на мобильных экранах.
3. Добавление навигации (стрелочки) лево/право.
4. Создание пагинации под слайдером (точки, обозначающие кол-во слайдов и подсвет активного слайда).
